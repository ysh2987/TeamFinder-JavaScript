# TeamFinder - JavaScript

## 프로젝트 소개 
- 전국 스포츠 팀 모집 서비스
- 기간 : 2021/12/13 ~ 2021/12/23


## 실행 방법
```
① 해당 레포지토리를 클론한다.
② 프로젝트의 패키지를 설치한다. (npm install)
④ scripts 명령어로 프로젝트를 실행한다. (npm start)
```
